# Домашние задания

В этой папке размещаются домашние задания по семинарам.

## Правила оформления

1. Для каждого задания создавайте отдельную папку: `HW01/`, `HW02/`, `HW03/` и т.д.
2. Основной файл задания должен называться так же, как папка:
   - `HW01/HW01.ipynb`
   - `HW02/HW02.ipynb`
   - и т.д.
3. Дополнительные файлы (вспомогательные скрипты, данные) можно размещать в той же папке.
4. **Не переименовывайте** папки `HWNN/` — это нарушит автоматическую проверку.

## Структура папки задания

```
HW01/
├── HW01.ipynb      # Основной ноутбук с решением
├── utils.py        # (опционально) вспомогательный код
└── data/           # (опционально) данные для задания
```

## Рекомендации

- Комментируйте код и объясняйте свои решения.
- Запускайте все ячейки ноутбука перед коммитом.
- Проверяйте, что ноутбук воспроизводится с чистого состояния.
