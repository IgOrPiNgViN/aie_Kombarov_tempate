# HW10-11 – Компьютерное зрение: CNN, transfer learning, сегментация

## 1. Кратко: что сделано

- Часть A: классификация CIFAR-100 — простая CNN без аугментаций (C1), с аугментациями (C2), ResNet18 head-only (C3) и partial fine-tune layer4+fc (C4).
- Часть B: сегментация OxfordIIITPet — pretrained DeepLabV3_ResNet50, два режима постобработки (V1 — базовый порог, V2 — морфологическая очистка).

## 2. Среда и воспроизводимость

- Python: 3.12
- torch / torchvision: 2.5.1+cu121 / 0.20.1+cu121
- Устройство (CPU/GPU): CUDA (если доступна), иначе CPU
- Seed: 42
- Как запустить: открыть `HW10-11.ipynb` и выполнить Run All.

## 3. Данные

### Часть A: CIFAR-100

- Датасет: CIFAR-100 (100 классов, 32×32 RGB), 8 000 train-примеров (subsample из 50 000)
- Разделение: 80/20 train/val из train-части + test из torchvision (10 000 примеров)
- Трансформации:
  - C1: ToTensor + Normalize(CIFAR mean/std)
  - C2: RandomHorizontalFlip + RandomCrop(32, padding=4) + ColorJitter + Normalize
  - C3/C4: Resize(64) + Normalize(ImageNet mean/std)

### Часть B: OxfordIIITPet

- Датасет: OxfordIIITPet (segmentation, test split, 3 669 изображений)
- Маски: trimap (1=foreground/pet, 2=border, 3=background)
- Foreground: класс 1 (животное) — выбран как основной foreground-сценарий, т.к. это главный объект на каждом изображении.

## 4. Часть A (S10): модели и обучение

- **SimpleCNN (C1/C2):** 3 свёрточных блока (Conv2d→BatchNorm→ReLU, 32→64→128 каналов) + AdaptiveAvgPool + FC(128→256→100). Dropout(0.3) в FC-голове.
- **ResNet18 head-only (C3):** pretrained ResNet18, все параметры заморожены кроме `fc = Linear(512, 100)`.
- **ResNet18 finetune (C4):** pretrained ResNet18, разморожены `layer4` + `fc`.
- Loss: CrossEntropyLoss
- Optimizer: Adam (lr=1e-3 для C1/C2/C3, lr=1e-4 для C4)
- Epochs: 8 (C1/C2), 6 (C3/C4)
- Batch size: 256 (C1/C2), 128 (C3/C4)

## 6. Часть B (S11): сегментация

- **V1 (basic):** порог 0.5 для бинаризации

## 7. Анализ

**Влияние аугментаций (C2 vs C1):** C2 показал чуть меньшую val_acc (0.1412 vs 0.1481). Это объяснимо: при малом числе эпох (8) и небольшом датасете (8 000 примеров) аугментации замедляют сходимость — модель не успевает «выучить» усложнённые варианты изображений. При большем числе эпох аугментации, как правило, дают прирост обобщающей способности.

**Transfer learning head-only (C3 vs C1/C2):** Переход к ResNet18 с замороженным backbone дал скачок точности более чем вдвое — с ~0.14 до 0.3719. ImageNet-признаки в backbone оказались достаточно информативными даже для CIFAR-100 с уменьшенным входом (64×64), что демонстрирует мощь предобученных представлений.

**Partial fine-tuning (C4 vs C3):** Разморозка `layer4` + `fc` с меньшим lr=1e-4 дополнительно подняла точность с 0.3719 до 0.4925 (+5.4 п.п. val, 48.5% test). При этом тренировочная точность C4 достигла 88%, что свидетельствует о хорошей обучаемости модели. Маленький lr при дообучении критичен: он позволяет плавно адаптировать pretrained-веса без их разрушения.

**Почему C4 — лучший выбор для CIFAR-100:** CIFAR-100 — задача с 100 классами, где признаки нижних слоёв (края, текстуры) универсальны, а верхние слои нужно адаптировать под конкретные категории. Именно такой подход реализует C4 — заморожены первые слои (быстрая и стабильная фильтрация), а `layer4` адаптируется к высокоуровневым паттернам CIFAR-100.

**Морфологическая постобработка (V2 vs V1):** V2 (порог 0.4 + closing + opening) дал несколько меньший mean_iou (0.6639 vs 0.6893) и чуть меньший recall (0.9815 vs 0.9857). Снижение порога до 0.4 увеличивает покрытие, однако последующая морфологическая очистка убирает мелкие детали, которые были частью разметки. В данном случае V1 точнее. Тем не менее V2 полезен при сценариях с шумными предсказаниями — он зачищает изолированные артефакты.

**Выбор foreground:** В OxfordIIITPet каждый кадр содержит одного питомца (кошку или собаку). DeepLabV3 обучен на PASCAL VOC, где есть классы `cat` и `dog` (и другие животные). Суммирование вероятностей по всем животным-классам PASCAL VOC обеспечивает наилучшее покрытие питомцев независимо от вида.

## 8. Итоговый вывод

Для задачи классификации CIFAR-100 рекомендуется конфиг **C4 (ResNet18 partial finetune)**: он даёт test accuracy ~48.5%, что в 3+ раза выше SimpleCNN. Ключ к успеху — использование pretrained-признаков ImageNet в сочетании с аккуратной адаптацией верхних слоёв (малый lr=1e-4). Простая CNN недостаточна для 100 классов при ограниченных данных и числе эпох.

Для сегментации OxfordIIITPet лучше показал **V1 (базовый порог 0.5)** с mean_iou=0.6893. Морфологическая постобработка (V2) снизила метрики в данном случае, так как маски животных достаточно компактны и не требуют сглаживания. В более зашумлённых задачах V2 был бы предпочтительнее.

**Направления для улучшения:** (1) Увеличить обучающую выборку до полных 50 000 примеров и число эпох для C4 — это позволит выйти на 60%+ accuracy. (2) Дообучить DeepLabV3 на OxfordIIITPet с маскированием триmap-граничных пикселей — ожидается рост mean_iou до 0.80+.
